# vuepress-plugin-sakana-widget

For vuepreess2.x

```js
plugins: [
    ['vuepress-plugin-sakana', {}]
]
```


```js
plugins: [
    ['vuepress-plugin-sakana', {
        config: {
            scale: 0.5,
            canSwitchCharacter: true,
        },
        styleObject: {
            position: "fixed",
            right: 0,
            bottom: "24px",
            "transform-origin": "100% 100%",
        }
    }]
]
```
